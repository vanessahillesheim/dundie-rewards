# Contributing to Dundie Project

Sumary of project

## Guidelines

- Backwards compatibility
- Multiplataform
- Python 3 only

## Code of Conduct

- Be gentle

## How to contibute

### Fork repository

- Click fork button on [github repo](https://github.com/vanessahillesheim/dundie-rewards)

### Clone to local dev envoirement

```bash
git clone https://github.com/vanessahillesheim/dundie-rewards
```

### Prepare virtual env

```bash
de dundie-rewards
make virtual env
make install
```

### Coding style

- This project follows PEP8

### Run tests

```bash
make test

```

### Commit rules

- We follow conventional commit messages ex: '[bugfix] reason #issue`
- We required signed commits

### Pull Request Rules

- We required all tests to be passing
